# CalliOpeNLP
CalliOpeNLP, is a new open-source python-based automated tool designed to support biomedical voice health and spoken language data collections. It offers a free, standalone automated collecton option which avoids unsecured data transfer useful in clinical voice biobank context.

The original CalliOpeNLP (v1) includes computer-generated spoken-text instructions for 18 clinically validated voice tasks. CalliOpeNLP automatically records, labels, and tracks patients' spoken compliance using low latency voice-to-text transcript analytics. This tool provides a new practical application enhancing automated voice data collection capabilities. A newer CalliOpeNLP (v2) also includes automatic extraction of Praat-based acoustic features, a voice report (i.e., this includes additional demographic data, Voice Health Index-10 survey, ASR transcript, task repeats, acoustic features) and Calliope auto-accent selection based on user's given nationality.

In ancient Greek mythology, Calliope was the 'Muse of Eloquence and Epic Poetry', and her name meant 'beatifully-voiced'. Hence, the wordplay on Calliope, open-source, and natural language processing (NLP).

Please, reference the following in manuscripts and/or presentations:

Stasak, B., Li, R., Chacon, A., Black, R., & Madill, C., 2026. CalliOpeNLP: A Standalone Digital Health Voice Data Collection Research Tool, In: Proc. INTERSPEECH 2026, Sydney, NSW – Australia, pp. 1–5.
