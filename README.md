# CalliOpeNLP
CalliOpeNLP, is a new open-source python-based automated tool designed to support biomedical voice health and spoken language data collections. It offers a free, standalone automated option which avoids unsecured data transfer.

The original CalliOpeNLP includes computer-generated spoken-text instructions for 18 clinically validated voice tasks. CalliOpeNLP automatically records, labels, and tracks patients' spoken compliance using low latency voice-to-text transcript analytics. This tool provides a new practical application enhancing automated voice data collection capabilities. A newer version (V2) also includes automatic extraction of Praat-based acoustic features per user.
